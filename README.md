# add_noise_to_photo
Цель работы показать как работают разные виды распределений (в данном случае хи квадрат, гауcса, равномерное, экспоненциальное, гамма, вейбулла) на примере добавления шума на изображение.

В папке static/uploads/ лежит пример обработанного изображения всеми представленными видами распредлений.  

Для оценки оценки качества фильтрации шума на фото принято использование таких метрик как, MSE, PSNR, SSIM. Чем выше PSNR и SSIM, тем лучше фильтр справляется с задачей, чем ниже MSE => тем лучше фильтр справляется с задачей.

Файл с расчитанными метриками называется metrics.csv 

на основе расчитанных метрика для каждого вида шума можно сделать следущие выводы:

1. 
    Для гауссова шума медианный фильтр справляется с задачей лучше всего, значения метрик по сравнению с другими фильтрами самые высокие (PSNR, SSIM) и самые низкие (MSE)
2. 
    Для равномерного шума медианный фильтр справляется с задачей лучше всего, значения метрик по сравнению с другими фильтрами самые высокие (PSNR, SSIM) и самые низкие (MSE)
3. 
    Для хи-квадрат шума медианный фильтр справляется с задачей лучше всего, значения метрик по сравнению с другими фильтрами самые высокие (PSNR, SSIM) и самые низкие (MSE)
4. 
    Для экспоненциального шума медианный фильтр справляется с задачей лучше всего, значения метрик по сравнению с другими фильтрами самые высокие (PSNR, SSIM) и самые низкие (MSE)
5. 
    Для гамма шума медианный фильтр справляется с задачей лучше всего, значения метрик по сравнению с другими фильтрами самые высокие (PSNR, SSIM) и самые низкие (MSE)
6. 
    Для вейбулла шума медианный фильтр справляется с задачей лучше всего, значения метрик по сравнению с другими фильтрами самые высокие (PSNR, SSIM) и самые низкие (MSE)

НО, стоит отметить что метрики сравнивают значения каждого пикселя с оригиналом, что показывает насколько фильтр близок к оригиналу, а не насколько картинка после его использования становиться понятной глазу

Если сравнивать визуальное воспритие то все выбранные картинки после медианного фильтра выглядят как мыло, что нельзя сказать про другие фильтры, которые судя по метрикам со своей задачей справились хуже, но визуальоне восприятие у них лучше. В качестве фаворита можно выделить Билатериальный фильтр, который лучше всего справляется с задачей сохранения краев и текстур(пример где это лучше всего заметно, это равномерное распредление).
