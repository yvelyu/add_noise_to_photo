# add_noise_to_photo
Цель работы показать как работают разные виды распределений (в данном случае хи квадрат, гауcса, равномерное, экспоненциальное, гамма, вейбулла) на примере добавления шума на изображение.

В папке static/uploads/ лежит пример обработанного изображения всеми представленными видами распредлений.
